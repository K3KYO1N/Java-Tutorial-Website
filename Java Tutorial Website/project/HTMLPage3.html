Recursion Algorithm

What is recursion?

Recursion is an algorithm that calls on itself. It is useful for solving problems that can be solved by repeating a process that involves itself.

import java.util.*;

public class Main{

public static void main(String[] args) {

System.out.println(fac(5));

}
static int fac(int i) {

if(i==1) return 1;
else return i * fac(i-1);

}

}

In the above example, the algorithm will print out the value of 5 factorial. The user defined "fac" method takes in a integer, and returns the factorial of the interger. The factorial of a number is equal to the number multiplied by the factorial of the number subtract one. Therefore, this method takes a number i, and multiplies it by the factorial of i-1 by calling on itself. A important thing about recursion is that it must have a base case, otherwise it will go into an infinite loop. The base case in the above example is that if i is 1, it will return 1, because 1 factorial is still 1.


Application of recursion | Greatest Common Divisor


import java.util.*;

public class Main{

public static void main(String[] args) {

System.out.println(GCD(30, 18));

}
static int GCD(int m, int n) {

if(n == 0) return m;
int rem = m%n;
m = n; n = rem;
return GCD(m, n);

}

}

The above example uses recursion as an application of the euclidean algorithm. The algorithm is used to find the greatest common divisor between two integers. The base case is that if the second integer is 0, it would return the first integer. Otherwise, the first integer will become the value of the second integer, and the second integer will become the value of the first integer(before it became the value of the second integer) modulo the second integer. The function will then call on itself until it hits the base case. Since the above example is using the numbers 30 and 18, it will produce the output of 6.

Basics of Java

Variables in Java

Variables are used to store data. There are different types of variables.
  • String : stores text.

  • int : stores integers.

  • float : stores decimals(can store 32 bits).

  • double : stores decimals(can store 64 bits).

  • char : stores a singular character.

  • boolean : stores either true or false.


Input and Output

To read user input in java, you would need to import the scanner class and create a scanner object. You will then be able to read input using your scanner.

import java.util.Scanner;

public class Main{

public static void main(String[] args) {

Scanner sc = new Scanner(System.in);
String temp = sc.nextLine();

}

}
In the above example, the value of temp will be your input.

To output something in java, you can use the print method.

import java.util.Scanner;

public class Main{

public static void main(String[] args) {

Scanner sc = new Scanner(System.in);
String temp = sc.nextLine();
System.out.println(temp);

}

}
In the above example, the value of temp will be your output.


For loops

For loops are used to repeat a line of code.

import java.util.Scanner;

public class Main{

public static void main(String[] args) {

int temp = 1;

for(int i=0; i<5; i++) temp++;
System.out.println(temp);

}

}

In the above example, temp has an initial value of 1. The int i represents the index of the for loop. It will start from i = 0, and loop until i = 4,
looping a total of 5 times. Each line, the value of temp is increased by 1. Therefore, the output value of temp will be 6.

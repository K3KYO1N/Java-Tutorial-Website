Custom Comparator

Comparing Strings


import java.util.*;

public class Main{

public static void main(String[] args) {

Scanner sc = new Scanner(System.in);
int n = Integer.parseInt(sc.nextLine());
String s[] = new String[n];
for(int i =0;i< n;i++) s[i]=sc.nextLine();
Arrays.sort(s, new com());
for(int i =0;i < n;i++) {
System.out.print(s[i]);
}

}
static class com implements Comparator<String>{

public int compare(String a, String b) {
String ab = a+b, ba = b+a;
return ab.compareTo(ba);
}

}

}

The above code takes in an integer n, then takes in n strings. The output will be a combination of the inputted string sorted from lowest to greatest in alphabetical order. The "com" function is a custom sort algorithm for strings. It takes in 2 strings, and compares them by creating two combinations of the strings(one with string a followed by string b, another with string b followed by string a). The above code utilizes java's Arrays.sort() function, implementing the custom sort fucntion, to sort an array of strings. For example, if the input is: 4 ac aa ad ab. The output would be aaabacad. Note: this algorithm can also sort numbers because it takes the string value of inputted numbers.
